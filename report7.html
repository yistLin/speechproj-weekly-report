End-to-End Conversational Model

Most of the value of deep learning today is in narrow domains where you can get a lot of data. Here’s one example of something it cannot do: have a meaningful conversation. There are demos, and if you cherry-pick the conversation, it looks like it’s having a meaningful conversation, but if you actually try it yourself, it quickly goes off the rails.

- Andrew Ng

Reinforcement Learning

  • Chatbots will take iterations and feedback loops to develop and to perfect

Dialog System

A typical activity cycle in a dialog system contains the following phases:

  • The text is analyzed by a natural language understanding (NLU) unit.
  • The semantic information is analyzed by the dialog manager
  • The output of the DM is usually converted to human language by the natural language generation (NLG) unit.
